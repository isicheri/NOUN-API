# NOUN Student Support Platform - Backend

This is the backend service for the **NOUN Student Support Platform**, designed to help students with admission guidance, document handling, and other academic support services.  
It is built with **NestJS** and **Prisma ORM**, and uses **Supabase** for database, real-time updates, and file storage.
